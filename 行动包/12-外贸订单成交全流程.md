# 外贸订单成交全流程

## 一、订单流程总览

### 1.1 完整订单流程

```
询盘 → 报价 → 样品 → 合同 → 生产 → 验货 → 出货 → 收款 → 售后
```

### 1.2 各阶段时间参考

| 阶段 | 正常周期 | 可缩短 | 可延长 |
|------|----------|--------|--------|
| 询盘响应 | 24小时 | 4小时 | 48小时 |
| 报价 | 1-3天 | 1天 | 7天 |
| 样品 | 7-15天 | 5天 | 30天 |
| 合同确认 | 1-3天 | 1天 | 7天 |
| 生产 | 15-45天 | 7天 | 90天 |
| 验货 | 2-5天 | 1天 | 10天 |
| 出货 | 3-7天 | 1天 | 15天 |
| 收款 | 30-90天 | T/T即期 | L/C远期 |

## 二、询盘处理

### 2.1 询盘分类

| 类型 | 特征 | 处理优先级 | 响应时间 |
|------|------|------------|----------|
| 真实需求型 | 有明确需求、数量、时间 | 高 | 即时 |
| 收集信息型 | 了解市场/对比价格 | 中 | 24小时 |
| 垃圾询盘 | 无实质内容 | 低 | 无需回复 |
| 钓鱼询盘 | 试探性索取 | 警惕 | 可忽略 |

### 3.2 报价策略

| 客户类型 | 策略 | 说明 |
|----------|------|------|
| 新客户 | 先报高价 | 留降价空间 |
| 竞争询价 | 参考对手价格 | 保持竞争力 |
| 长期客户 | 适当优惠 | 维护关系 |
| 大客户 | 阶梯报价 | 量大从优 |
| 定制需求 | 成本+利润 | 确保利润 |

### 3.3 报价跟进

```markdown
## 报价跟进计划

Day 1: 发送报价
Day 3: 第一次跟进
  - 询问收到情况
  - 解答疑问

Day 7: 第二次跟进
  - 强调产品优势
  - 提醒有效期

Day 14: 第三次跟进
  - 询问决策进展
  - 提供替代方案

Day 30: 最后跟进
  - 确认是否继续
  - 保持联系

### 5.2 付款方式选择

| 付款方式 | 优点 | 缺点 | 适用场景 |
|----------|------|------|----------|
| T/T 30/70 | 安全，主流 | 客户资金压力大 | 大额订单 |
| T/T 即期 | 安全，快捷 | 客户风险高 | 小额/老客户 |
| L/C 即期 | 银行担保 | 费用高，复杂 | 新客户大单 |
| L/C 远期 | 资金灵活 | 风险较高 | 特殊安排 |
| PayPal | 方便 | 手续费高 | 小额样品 |

```
