# 目标市场深度调研模板

## 一、调研框架

### 1.1 调研目的

```
核心问题：
1. 这个市场值得进入吗？
2. 我们能在这个市场成功吗？
3. 我们应该如何进入这个市场？
```

### 1.2 调研维度

| 维度 | 内容 | 优先级 |
|------|------|--------|
| 市场规模 | 潜在容量、增长趋势 | 高 |
| 竞争分析 | 现有玩家、竞争格局 | 高 |
| 客户特征 | 需求特点、采购习惯 | 高 |
| 进入门槛 | 法规要求、本地化要求 | 中 |
| 渠道机会 | 可用渠道、获客成本 | 中 |
| 风险评估 | 政治、经济、汇率风险 | 中 |

---

## 二、市场概况调研

### 2.1 基本信息表

```markdown
## [市场名称] 市场概况

### 基本数据
- 国家/地区：[填写]
- 人口：[X亿/百万]
- GDP：[X万亿美元]
- 人均GDP：[$X]
- 年增长率：[X%]

### 贸易关系
- 与中国的贸易额：[X亿美元]
- 贸易政策：[关税、壁垒等]
- 主要进口产品：[列出]

### 商业环境
- 营商环境排名：[世界银行排名]
- 清关难度：[简单/中等/困难]
- 腐败程度：[低/中/高]
- 语言：[官方语言]
- 支付习惯：[TT/信用证/其他]
```

### 2.2 数据来源

| 数据类型 | 来源 |
|----------|------|
| 宏观数据 | World Bank, IMF, UN |
| 贸易数据 | 中国海关, UN Comtrade |
| 行业报告 | Statista, Grand View Research |
| 竞品分析 | 行业报告, 竞品官网 |
| 客户调研 | 问卷, 访谈 |

## 三、目标客户调研

### 3.1 客户画像模板

```markdown
## [目标客户] 画像

### 基本特征
- 公司类型：[分销商/制造商/电商/工程商]
- 规模：[年营业额 X-X 百万美元]
- 员工数量：[X-X 人]
- 地理分布：[城市/地区]

### 采购特征
- 采购周期：[月度/季度/年度]
- 决策人：[采购经理/老板/工程师]
- 采购渠道：[线上/线下/展会]
- 付款方式：[TT/LC/赊账]

### 需求特点
- 关注点：[价格/质量/交期/服务]
- 痛点：[当前面临的问题]
- 偏好：[品牌/定制/现货]

### 触达方式
- 主要平台：[B2B平台/谷歌/社媒]
- 常用语言：[英语/本地语言]
- 最佳联系时间：[工作时间]
```

### 3.2 调研方法

| 方法 | 适用场景 | 成本 |
|------|----------|------|
| 问卷调查 | 大量样本 | 低 |
| 电话访谈 | 深度了解 | 中 |
| 展会接触 | 面对面交流 | 中高 |
| 社媒调研 | 了解需求 | 低 |
| 竞品客户 | 了解需求 | 低 |
