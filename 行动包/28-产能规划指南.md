# 产能规划指南

## 一，产能规划的重要性

### 1.1 什么是产能规划

```
产能规划 = 确保生产能力匹配市场需求

核心问题：
- 能不能生产出来？
- 能不能按时交付？
- 能不能保证质量？
- 能不能控制成本？
```

### 1.2 产能规划的意义

```
产能规划的价值：
- 确保订单按时交付
- 避免产能过剩
- 控制生产成本
- 提升客户满意度
- 抓住市场机会
```

## 六，产能风险管理

### 6.1 风险识别

```markdown
## 产能风险识别

### 需求风险
| 风险 | 影响 | 概率 | 应对方案 |
|------|------|------|----------|
| 订单超预期 | 产能不足 | 中 | 加班/外协 |
| 订单低于预期 | 产能过剩 | 中 | 调整生产/开发新客户 |
| 订单波动大 | 生产不稳定 | 高 | 建立安全库存 |

### 供应风险
| 风险 | 影响 | 概率 | 应对方案 |
|------|------|------|----------|
| 原材料供应不足 | 停产 | 低 | 多供应商 |
| 设备故障 | 产能下降 | 中 | 维护保养/备机 |
| 人员短缺 | 产能下降 | 中 | 储备人员 |

### 交付风险
| 风险 | 影响 | 概率 | 应对方案 |
|------|------|------|----------|
| 交期延误 | 客户流失 | 高 | 提前预警/加急 |
| 质量问题 | 返工重做 | 中 | 品控强化 |
| 物流延误 | 交付延迟 | 中 | 多物流方案 |
```

### 6.2 风险应对

```markdown
## 风险应对策略

### 产能不足应对
1. 加班生产
2. 增加临时工
3. 外协加工
4. 调整交期
5. 客户沟通

### 产能过剩应对
1. 开发新客户
2. 增加产品线
3. 调整生产计划
4. 人员培训
5. 设备维护

### 紧急订单应对
1. 评估可行性
2. 调整生产计划
3. 加班加点
4. 外协支持
5. 客户沟通
```
