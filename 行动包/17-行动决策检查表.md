# 行动决策检查表

## 一，决策检查的意义

### 1.1 为什么需要检查表

```
决策前检查 = 减少冲动决策 = 降低风险

没有检查的问题：
- 冲动行动
- 考虑不周
- 风险忽视
- 后悔决策
```

### 1.2 检查表的价值

| 检查前 | 检查后 |
|--------|--------|
| 冲动决策 | 理性决策 |
| 考虑片面 | 全面评估 |
| 风险忽视 | 风险可控 |
| 盲目行动 | 有据可依 |

## 二，行动前必问问题

### 2.1 目的确认

```markdown
## 目的确认检查

□ 这个行动的目标是什么？
□ 这个目标具体、可衡量吗？
□ 这个目标真的重要吗？
□ 这个目标和其他目标冲突吗？
□ 不做这个行动会怎样？

**目的检查结论：**
□ 目标清晰，值得行动
□ 目标模糊，需要明确
□ 目标不重要，取消行动
```

### 2.2 风险评估

```markdown
## 风险评估检查

### 最坏情况
- 最坏情况是什么？________
- 发生的可能性有多大？□高 □中 □低
- 这个结果我能承受吗？□能 □不能

### 风险应对
- 如果出现问题，我能做什么？________
- 我需要准备什么后备方案？________
- 我需要告知谁？________

**风险检查结论：**
□ 风险可控，可以行动
□ 风险较高，需要降低
□ 风险太大，取消行动
```

### 2.3 资源确认

```markdown
## 资源确认检查

### 所需资源
- 资金：需要$____，我有$____
- 时间：需要____小时，我有____小时
- 人员：需要____人，我有____人
- 技能：需要____，我会____

### 资源差距
- 资金差距：□无 □有（$____）
- 时间差距：□无 □有（____小时）
- 人员差距：□无 □有（____人）
- 技能差距：□无 □有（____）

**资源检查结论：**
□ 资源充足，可以行动
□ 资源不足，需要补充
□ 资源严重不足，暂缓行动
```

## 三，决策检查表

### 3.1 标准检查表

```markdown
## 行动决策标准检查表

行动描述：____________
决策时间：____________

### 目的检查（必须全部通过）
□ 目标具体明确
□ 目标可衡量
□ 目标有价值
□ 目标与战略一致
□ 不行动的代价更大

### 风险检查（必须全部通过）
□ 最坏情况已知
□ 风险可承受
□ 有应对方案
□ 不会造成不可逆损失

### 资源检查（必须全部通过）
□ 资金充足或有保障
□ 时间足够
□ 人员到位
□ 技能具备

### 时机检查
□ 时机合适
□ 不是冲动决策
□ 不是情绪决策

### 替代方案检查
□ 考虑过其他方案
□ 比较过各方案优劣
□ 当前方案是最优选择

### 决策建议
□ 执行
□ 暂缓执行
□ 不执行
```

### 3.2 快速检查表（5分钟决策）

```markdown
## 快速行动检查表（5分钟版）

用"是/否"快速回答：

1. 目标清晰吗？□是 □否
2. 最坏结果能承受吗？□是 □否
3. 有足够资源吗？□是 □否
4. 时机合适吗？□是 □否
5. 比其他方案更好吗？□是 □否

**结果：**
- 5个"是"：立即执行
- 3-4个"是"：谨慎执行
- <3个"是"：重新考虑
```

## 六，常见决策陷阱

### 6.1 决策陷阱清单

```markdown
## 决策陷阱清单

### 认知陷阱
□ 确认偏误 - 只看到支持自己观点的信息
□ 锚定效应 - 被第一个信息过度影响
□ 过度自信 - 高估自己判断准确性
□ 损失厌恶 - 对损失的反应过度强烈

### 情感陷阱
□ 冲动决策 - 情绪激动时做决定
□ 从众心理 - 因为别人在做所以做
□ 避免后悔 - 为避免后悔而做决定
□ 社会认同 - 追求认可而做决定

### 信息陷阱
□ 信息过载 - 信息太多无法判断
□ 信息不足 - 信息太少盲目决策
□ 沉没成本 - 因为投入多而继续
□ 事后诸葛 - 事后认为当初可预测

### 避免方法
- 引入第三方视角
- 设定冷静期
- 量化分析
- 检查决策清单
```

### 6.2 避免陷阱检查

```markdown
## 避免决策陷阱检查

□ 我是基于数据而不是感觉做决定
□ 我考虑过相反的证据
□ 我没有因为别人在做就做
□ 我没有被情绪影响
□ 我没有因为投入多就继续
□ 我设定了冷静期
□ 我量化分析了利弊
□ 我征求了第三方意见

**陷阱检查结果：**
- 全部通过：决策质量高
- 少数未通过：需谨慎
- 大多数未通过：重新决策
```
