# 最小行动启动清单

## 一，为什么要"最小行动"

### 1.1 最小的力量

```
最小行动 = 打破拖延的第一步 = 开启行动的钥匙

问题：想太多、做太少
解法：最小化行动，降低启动门槛

原理：
- 大任务 → 畏惧 → 拖延
- 小任务 → 简单 → 行动
- 一旦开始 → 惯性 → 继续
```

### 1.2 最小行动原则

| 原则 | 说明 |
|------|------|
| 越小越好 | 能多小就多小 |
| 立即可做 | 不需要准备 |
| 具体明确 | 知道做什么 |
| 有意义 | 和目标相关 |
| 可累积 | 能连接下一步 |

## 五，克服拖延技巧

### 5.1 2分钟法则

```
2分钟能做完的事 → 立即做

示例：
- 发1封邮件 → 2分钟
- 填1个表格 → 2分钟
- 回1条消息 → 2分钟
- 看1篇文章 → 2分钟

如果一件事2分钟内能做完 → 立即做
如果不能 → 拆成2分钟的小步骤
```

### 5.2 5秒法则

```
倒数5秒 → 立即行动

应用场景：
- 不想起床 → 5-4-3-2-1-起来
- 不想打电话 → 5-4-3-2-1-打
- 不想发邮件 → 5-4-3-2-1-发

原理：
- 倒数给大脑"准备信号"
- 跳过犹豫，直接行动
```
