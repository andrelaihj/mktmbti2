# 机会筛选与聚焦指南

## 一、为什么要筛选机会

### 1.1 机会太多是陷阱

```
陷阱1：什么都想做
- 今天亚马逊
- 明天独立站
- 后天TikTok
- 结果：什么都没做成

陷阱2：追风口
- 听到什么火就做什么
- 永远在跟风
- 结果：永远慢半拍

陷阱3：选择困难
- 觉得每个机会都好
- 不知道怎么选
- 结果：迟迟不动
```

### 1.2 聚焦的力量

| 聚焦前 | 聚焦后 |
|--------|--------|
| 分散精力 | 集中资源 |
| 每个都浅尝辄止 | 一个深耕到底 |
| 永远从零开始 | 积累持续增长 |
| 什么都有一点 | 某个领域最强 |

## 二、机会筛选框架

### 2.1 筛选三维度

```
维度1：匹配度
└── 这个机会和你的能力/资源匹配吗？

维度2：吸引力
└── 这个机会的市场够大、增长够快吗？

维度3：可行性
└── 你能在合理时间内做到吗？
```

## 六、决策框架

### 6.1 机会选择决策树

```
问题1：这个机会和你的核心能力匹配吗？
├── 否 → 放弃，寻找更匹配的
└── 是 → 继续问题2

问题2：这个机会的市场够大吗？（年增长>10%）
├── 否 → 放弃，寻找更大市场
└── 是 → 继续问题3

问题3：你能在3-6个月内看到初步成果吗？
├── 否 → 降低预期或放弃
└── 是 → 继续问题4

问题4：你能承受失败的后果吗？
├── 否 → 放弃或降低投入
└── 是 → 可以尝试
```

## 八、常见问题

### Q1：同时看几个机会可以吗？

不建议。资源分散=每个都做不好。

**建议**：选择一个最匹配的，全力以赴。如果3个月后验证不行，再换。

### Q2：选错了怎么办？

**处理方式**：
1. 给自己3个月试用期
2. 设定明确的成功标准
3. 每月复盘进展
4. 达不到标准果断调整

### Q3：担心错过风口？

**认知调整**：
- 风口年年有，错过一个还有下一个
- 深耕一个领域，你会成为下一个风口
- 机会是给有准备的人的
