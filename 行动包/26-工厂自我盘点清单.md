# 工厂自我盘点清单

## 一，盘点的重要性

### 1.1 为什么要盘点

```
盘点 = 认清自己 = 找准方向

盘点价值：
- 知道自己有什么
- 知道自己的优势
- 知道自己的短板
- 找到出海的方向
- 避免盲目行动
```

### 1.2 盘点核心问题

```
盘点要回答的问题：

1. 我有什么？（资源盘点）
2. 我能做什么？（能力盘点）
3. 我缺什么？（差距盘点）
4. 我该怎么补？（行动计划）
```
