# 代运营合作模板

## 一，代运营概述

### 1.1 什么是代运营

```
代运营 = 委托专业团队运营你的出海业务

价值：
- 专业人做专业事
- 节省时间和精力
- 快速启动和见效
- 降低试错成本

风险：
- 服务质量参差不齐
- 可能不够了解产品
- 沟通成本
- 依赖风险
```

### 1.2 代运营类型

```markdown
## 代运营类型

### 平台代运营
□ 阿里国际站代运营
□ 亚马逊代运营
□ 独立站代运营

### 渠道代运营
□ 谷歌广告代运营
□ Facebook广告代运营
□ 社媒代运营

### 全方位代运营
□ 整体出海业务托管
□ 团队外包
□ 利润分成
```

## 三，合作模式

### 3.1 常见合作模式

```markdown
## 常见合作模式

### 模式1：纯服务费
- 固定月费：$1000-5000/月
- 适合：有一定基础的
- 优点：成本可控
- 缺点：服务商动力可能不足

### 模式2：底薪+提成
- 底薪：$500-2000/月
- 提成：销售额1-5%
- 适合：成长型业务
- 优点：利益绑定
- 缺点：提成计算复杂

### 模式3：纯提成
- 提成：销售额3-10%
- 适合：风险共担
- 优点：服务商动力足
- 缺点：服务商可能挑客户

### 模式4：项目制
- 按项目收费
- 适合：特定需求
- 优点：目标明确
- 缺点：连续性差
```

### 3.2 模式选择建议

```markdown
## 模式选择建议

### 建议模式
□ 新手期：纯服务费（低成本试水）
□ 成长期：底薪+提成（利益绑定）
□ 成熟期：纯提成或项目制（专业分工）

### 选择考量
- 预算多少？
- 期望目标？
- 风险承受？
- 配合模式？
```

### 合同基本信息

- 签订日期：[日期]

□ 店铺日常运营
□ 产品上架优化
□ 询盘处理跟进
□ 数据分析报告

#### 营销推广
□ 广告投放管理
□ 推广方案制定
□ 效果监测优化
□ 月度推广报告

#### 内容运营
□ 图片处理优化
□ 文案撰写
□ 活动策划
□ 其他内容服务

### 服务标准
□ 每月产品上架不少于[X]个
□ 询盘回复时间不超过[X]小时
□ 每周提交数据报告
□ 每月提供运营分析

### 费用及支付
□ 服务费用：$____/月
□ 支付方式：[月付/季付]
□ 支付时间：每月[X]日前
□ 银行账户信息：[填写]

### 双方责任

#### 甲方责任
□ 提供产品信息和资料
□ 提供必要的账号权限
□ 配合乙方工作
□ 按时支付费用

#### 乙方责任
□ 按约定提供服务
□ 保护甲方信息安全
□ 定期汇报工作
□ 保守商业秘密

### 考核指标
□ 询盘数量目标：[X]个/月
□ 询盘转化目标：[X]%
□ 销售额目标：[X]/月（可选）
□ 其他指标：[填写]

### 保密条款
双方应对合作中获得的信息保密，未经对方书面同意，不得向第三方披露。

### 违约责任
□ 甲方逾期付款，每逾期一天按应付款的[X%]支付违约金
□ 乙方未完成约定服务，甲方有权要求退还相应费用
□ 任何一方违反保密义务，应赔偿对方损失

### 合同变更与终止
□ 合同期满前[X]天，双方可协商续约
□ 任何一方可提前[X]天书面通知终止合同
□ 合同终止后，乙方应配合完成交接工作

### 争议解决
本合同适用中华人民共和国法律。如有争议，协商解决；协商不成的，提交[仲裁/法院]解决。

### 签署

乙方（盖章）：____________ 日期：____________

## 六，合作管理

### 6.1 日常管理

```markdown
## 代运营日常管理

### 沟通机制
□ 每日：企业微信/微信群沟通
□ 每周：周例会（线上/线下）
□ 每月：月度考核和复盘
□ 随时：问题反馈和处理

### 会议机制
□ 周会内容：本周工作、下周计划、问题反馈
□ 月会内容：月度总结、考核评估、下月计划
□ 记录留存：会议纪要存档

### 文档管理
□ 工作进度文档
□ 数据报告文档
□ 沟通记录文档
□ 合同协议文档
```

### 6.2 问题处理

```markdown
## 问题处理流程

### 问题分类
□ 一般问题：2天内解决
□ 重要问题：及时沟通解决
□ 紧急问题：立即响应

### 处理流程
1. 问题发现和记录
2. 反馈给代运营
3. 代运营分析原因
4. 制定解决方案
5. 执行解决
6. 验证效果
7. 总结经验

### 问题升级
- 一般问题：运营负责人处理
- 重要问题：双方高层处理
- 严重问题：启动合同条款
```

## 七，合作退出

### 7.1 退出情形

```markdown
## 退出情形

### 主动退出（甲方）
□ 连续3个月考核不合格
□ 代运营严重违约
□ 合同到期不再续约
□ 公司战略调整

### 被动退出（乙方）
□ 甲方严重违约
□ 合作无法继续
□ 乙方业务调整

### 友好退出
□ 提前[X]天通知
□ 完成当前工作交接
□ 结算未付款项
□ 配合资料移交
```

- [ ] 持续优化改进

- [ ] 评估合作效果
- [ ] 总结合作经验
- [ ] 完成交接工作
- [ ] 结算相关费用
