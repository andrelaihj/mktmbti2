# 核心赛道定位指南

## 一、什么是赛道定位

### 1.1 赛道定位的定义

```
赛道 = 产品 + 市场 + 客户 的组合

好的赛道定位：
- 不是"我做什么产品"
- 而是"我为谁、提供什么产品、解决什么问题"

示例对比：
❌ "我们做阀门"
✓ "我们做北美工业水处理厂的特种阀门"
```

### 1.2 为什么要明确定位

| 没有定位 | 有定位 |
|----------|--------|
| 和所有人竞争 | 和少数人竞争 |
| 价格战 | 差异化 |
| 客户记不住你 | 客户一想到这个需求就想到你 |
| 什么客户都有 | 精准客户多 |
| 利润越来越薄 | 利润稳定甚至更高 |

## 二、定位三要素

### 2.1 定位三角形

```
        产品
       /    \
      /      \
     /   客户   \
    /    /  \    \
   /    /    \    \
 ------   市场   ------
```

**产品**：你卖什么？（具体的产品品类/特性）
**市场**：你卖给谁？（目标国家/地区/行业）
**客户**：解决什么问题？（客户的痛点和需求）

### 2.2 定位公式

```
定位 = [产品] + [目标市场] + [为谁解决什么问题]

示例：
- 特种阀门 + 北美 + 工业水处理厂的高压防腐需求
- 工业零部件 + 德国 + 汽车配件商的精密加工需求
- 户外家具 + 澳大利亚 + 别墅主人的耐用美观需求
```

## 三、定位分析方法

### 3.1 产品维度分析

```markdown
## 产品维度分析

### 产品分类
□ 整机产品
□ 零部件/配件
□ 原材料
□ 半成品

### 产品特性
□ 标准化产品
□ 定制化产品
□ 技术型产品
□ 价格型产品

### 产品深度
□ 单品突破（一款产品打天下）
□ 产品线覆盖（多个相关产品）
□ 解决方案型（产品+服务）

### 你的优势产品
1. [产品1] - 优势：[为什么能做]
2. [产品2] - 优势：[为什么能做]
3. [产品3] - 优势：[为什么能做]
```

### 3.2 市场维度分析

```markdown
## 市场维度分析

### 地理维度
□ 成熟市场（美/欧/日） - 门槛高，利润高
□ 新兴市场（东南亚/中东） - 增长快，竞争增加
□ 发展中市场（非洲/南美） - 门槛低，利润低

### 市场容量
- 目标市场的[品类]年市场规模：[X万美元]
- 年增长率：[X%]
- 中国供应商占比：[X%]

### 市场趋势
- [趋势1]：影响[正面/负面]
- [趋势2]：影响[正面/负面]
- [趋势3]：影响[正面/负面]
```

### 3.3 客户维度分析

```markdown
## 客户维度分析

### 客户类型
□ 品牌商/进口商 - 量大，稳定
□ 经销商/分销商 - 渠道依赖
□ 制造商 - 定制需求
□ 电商卖家 - 起量快
□ 工程商/项目商 - 项目制

### 客户规模
□ 大客户（年采购>100万）
□ 中客户（年采购10-100万）
□ 小客户（年采购<10万）

### 客户痛点
- 痛点1：[描述] - [你能解决吗]
- 痛点2：[描述] - [你能解决吗]
- 痛点3：[描述] - [你能解决吗]
```

## 四、定位策略

### 4.1 垂直定位策略

```
定义：在某个细分领域做深做透

示例：
❌ 做"工业阀门"
✓ 做"北美泳池SPA行业的温泉阀门"

适用场景：
- 资源有限，无法和大型玩家正面竞争
- 有某个细分领域的专业积累
- 目标市场有明显的细分需求
```

### 4.2 地理定位策略

```
定义：专注某个特定市场

示例：
❌ 做"欧洲市场"
✓ 做"德国汽车工业配套"

适用场景：
- 有某个市场的客户积累
- 有语言或本地化优势
- 某个市场竞争相对较小
```

### 4.3 应用定位策略

```
定义：专注某个应用场景

示例：
❌ 做"LED灯"
✓ 做"户外景观照明"

适用场景：
- 产品有多种应用
- 某个应用场景有特殊需求
- 你对这个应用有深入理解
```

### 4.4 客群定位策略

```
定义：专注某类客户

示例：
❌ 做"家具"
✓ 做"澳大利亚高端民宿业主"

适用场景：
- 有某类客户的资源
- 这类客户有共性需求
- 能为这类客户提供差异化价值
```

## 五、定位验证

### 5.1 验证问题清单

| 问题 | 回答 |
|------|------|
| 你的目标客户是谁？ | [具体描述] |
| 客户有什么痛点？ | [列出] |
| 你能解决吗？ | [能/不能] |
| 客户愿意为此付费吗？ | [愿意/不愿意] |
| 你比竞争对手强在哪？ | [列出] |
| 这个市场够大吗？ | [够/不够] |
| 你能持续提供价值吗？ | [能/不能] |

### 5.2 定位检验表

```markdown
## 定位检验

### 清晰性检验
□ 能否用一句话说清楚你的定位？
□ 10秒钟内能让人理解吗？
□ 客户一听就知道你是做什么的吗？

### 差异化检验
□ 这个定位有别人在做吗？
□ 你和竞争对手有什么不同？
□ 客户为什么要选你不选别人？

### 可行性检验
□ 你有能力服务这个定位的客户吗？
□ 客户能通过什么渠道找到你？
□ 这个市场能支撑你的业务吗？

### 持续性检验
□ 这个定位能让你长期发展吗？
□ 能建立竞争壁垒吗？
□ 未来有扩展空间吗？
```

## 六、定位执行

### 6.1 定位陈述模板

```markdown
## 定位陈述（1分钟版本）

我们是[公司名]，
专注于[目标市场/客户类型]的[产品品类]。

我们帮助客户解决[核心问题/痛点]，
通过[核心优势/差异化价值]。

与市场上其他供应商不同，我们[独特能力]。

联系我们：[联系方式/渠道]
```

### 6.2 定位落地检查

```markdown
## 定位落地检查

### 产品层面
□ 产品线围绕定位设计
□ 产品资料突出定位特点
□ 定价符合定位（高端/中端/低端）

### 渠道层面
□ 选择符合定位的平台/渠道
□ 投放针对定位客户的广告
□ 参展选择相关行业展会

### 内容层面
□ 公司介绍围绕定位
□ 产品描述突出定位
□ 社媒内容聚焦定位

### 团队层面
□ 团队理解定位
□ 销售话术符合定位
□ 客服标准匹配定位
```

## 七、常见问题

### Q1：定位太窄会不会没客户？

**相反**，定位越窄越精准：
- 客户容易找到你
- 客户需求你都能满足
- 容易建立专业形象
- 议价能力更强

### Q2：定位后还能改吗？

可以，但需要时间调整。建议：
- 初期定位后坚持6-12个月
- 根据数据调整细节
- 大的方向改变需要全面评估

### Q3：客户问"你们还做什么"怎么办？

**回答方式**：
"我们专注做[定位]，在这个领域我们有[X]年的经验...[简短说明优势]，您目前的[需求]，正是我们最擅长的。"
