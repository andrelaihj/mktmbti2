# 出海第一步行动指南

## 一，出海第一步的核心

### 1.1 为什么要"第一步"

```
出海第一步 = 打破从0到1的障碍

核心问题：
- 不知道从哪里开始
- 担心准备不够
- 害怕犯错
- 习惯性拖延

第一步的意义：
- 打破完美主义
- 建立信心
- 开启正向循环
- 减少焦虑
```

### 1.2 第一步原则

```
第一步原则：
1. 越小越好（降低门槛）
2. 立即可做（消除拖延）
3. 具体明确（知道做什么）
4. 有意义（和目标相关）
```

## 五，第一步常见问题

### 5.1 不知道从哪里开始

```
解决方案：
1. 从最小的一步开始
2. 先完成一个动作
3. 不要想太多，做起来再说
```

### 5.2 担心准备不够

```
解决方案：
1. 告诉自己：先完成再完美
2. 告诉自己：边做边学
3. 告诉自己：没有人一开始就很厉害
```

### 5.3 害怕犯错

```
解决方案：
1. 告诉自己：犯错是最好的学习
2. 告诉自己：小的犯错成本低
3. 告诉自己：不行动才是最大的错
```

### 5.4 习惯性拖延

```
解决方案：
1. 使用2分钟法则
2. 使用5秒法则
3. 降低门槛，从小开始
```
