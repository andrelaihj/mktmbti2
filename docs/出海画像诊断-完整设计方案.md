# 出海画像诊断 - 完整设计方案

## 一、诊断维度说明

### 四维评估模型

| 维度 | 英文 | 场景 | 题目数量 | 核心评估点 |
|------|------|------|---------|-----------|
| 产能底气 | capacity | 资源盘点 | 5题 | 生产能力、产品形态、供应链、出海资质、驱动力 |
| 市场嗅觉 | market | 市场洞察 | 6题 | 客户来源、跨境经验、市场关注度、目标用户了解度、进入门槛认知、产品准备度 |
| 行动惯性 | action | 行动记录+决心测试 | 6题 | 展会经验、出海准备、线上尝试、决心程度、跟进方式 |
| 决策算账 | decision | 风险意识 | 5题 | 资金认知、投入意愿、成交跟进、回报预期、决策依据 |

### 题目汇总（共22题）

```
资源盘点（产能底气）：
  Q14. 目前产能利用率在什么水平？
  Q16. 你卖的是产品还是一站式解决方案？
  Q17. 工厂在供应链上最常遇到的是哪类问题？
  Q_new_cap_1. 你的产品出口海外，以下哪些已经具备了？
  Q_new_cap_2. 你想出海的主要原因是什么？

市场洞察（市场嗅觉）：
  Q3. 你目前的客户主要来自哪些地区？
  Q4. 你有正在运营的跨境电商平台店铺吗？
  Q5. 你关注过哪些海外市场？
  Q6. 你知道目标市场的客户更看重什么吗？
  Q7. 你了解海外目标市场的进入门槛吗？
  Q8. 产品出口海外还缺什么？

行动记录+决心测试（行动惯性）：
  Q1. 你参加过国际展会或行业论坛吗？
  Q2. 你做过哪些出海准备工作？
  Q9. 你尝试过通过线上方式接触海外客户吗？
  Q10. 你尝试过哪些方式接触海外市场？
  Q20. 能接受多久才能看到海外订单？
  Q21. 出海项目你会如何跟进？

风险意识（决策算账）：
  Q11. 你知道开拓海外市场需要多少启动资金吗？
  Q12. 如果试水海外市场，你愿意投入多少？
  Q24. 有询盘但没有成交，你会怎么做？
  Q25. 你期望多久能收回出海投入的成本？
  Q26. 你平时做决定主要靠什么？
```

### 画像判定算法（优先级顺序）

```javascript
export const calculateProfile = (dimensions) => {
  const { capacity, market, action, decision } = dimensions;

  // 1. 六边形战士: 四维均在60以上
  if (capacity >= 60 && market >= 60 && action >= 60 && decision >= 60) {
    return 'E';
  }

  // 2. 制造巨匠: 产能强，市场和行动弱
  if (capacity >= 70 && market < 50 && action < 50) {
    return 'A';
  }

  // 3. 趋势探索者: 市场嗅觉强，产能弱
  if (market >= 70 && capacity < 50 && action < 50) {
    return 'B';
  }

  // 4. 破局先锋: 行动力强，决策弱
  if (action >= 70 && decision < 50 && market < 50) {
    return 'C';
  }

  // 5. 谋定后动: 决策强，行动弱
  if (decision >= 70 && action < 50) {
    return 'D';
  }

  // 6. 充分准备者: 产能和市场都强，行动弱
  if (capacity >= 70 && market >= 70 && action < 50) {
    return 'K';
  }

  // 7. 远见规划者: 市场强，行动弱
  if (market >= 60 && action < 50) {
    return 'H';
  }

  // 8. 热血冒险家: 市场和行动都强，产能弱
  if (market >= 70 && action >= 70 && capacity < 50) {
    return 'N';
  }

  // 9. 敏捷猎手: 行动强，市场强，决策弱
  if (action >= 60 && market >= 60 && decision < 50) {
    return 'G';
  }

  // 10. 稳健守护者: 产能强，决策强，市场和行动弱
  if (capacity >= 60 && decision >= 60 && market < 50 && action < 50) {
    return 'I';
  }

  // 11. 精算智多星: 决策强，其他中等
  if (decision >= 60 && capacity >= 50 && market >= 50 && action < 50) {
    return 'M';
  }

  // 12. 顺势弄潮: 行动强，其他弱
  if (action >= 60 && capacity < 50 && market < 50 && decision < 50) {
    return 'J';
  }

  // 13. 渴望探索者: 全都低
  if (capacity < 50 && market < 50 && action < 50 && decision < 50) {
    return 'L';
  }

  // 14. 初露锋芒: 决策中等，其他弱
  if (capacity < 50 && market < 50 && action < 50 && decision >= 50) {
    return 'O';
  }

  // 15. 蓄势待发: 默认类型（四维中等）
  return 'F';
};
```

## 三、每种画像详细描述（MBTI话术风格）

### A. 制造巨匠 🏭

**称号**: 匠心独运

**summary**: 专注制造，用产品力说话

**特征描述**:
> 你是那种把产品做到极致的人。工厂的每一道工序、每一个细节，你都精益求精。你的产品良品率高，客户口碑好，供应链也管理得井井有条。你相信好的产品自己会说话，与其花时间搞营销，不如把品质做到无可挑剔。现在，你的产品已经具备了出海的实力，只需要找到让世界看见它的方式。

**优势**:
- 产品质量精益求精
- 生产管理有条不紊
- 供应链稳定可靠
- 成本控制能力出众

**天赋课题**: 如何让好产品被更多人看见

**机会路径**: 你的品质是你最大的底气。现在需要的是让全球买家找到你的渠道。

**具体动作**:
1. 制作一套专业的英文产品手册，让品质可视化
2. 在国际B2B平台建立店铺，让客户主动找上门
3. 参加一次行业展会，面对面展示产品力

**行动包**: 《产品出海展示手册模板》、《国际B2B平台入驻指南》、《展会备战 checklist》

---

### B. 趋势探索者 🔮

**称号**: 洞若观火

**summary**: 洞察趋势，等待时机

**特征描述**:
> 你有一种特别的能力——总能比别人更早发现市场机会。你关注全球贸易动态，研究不同国家的消费趋势，了解什么产品在海外正在兴起。你的脑子里装着一个清晰的出海地图，只是还在等待一个合适的时机。你相信好的战略比盲目的行动更重要，宁可多看几步，也不愿贸然出击。

**优势**:
- 对市场趋势敏锐
- 善于观察和思考
- 信息整合能力强
- 有全局视野

**天赋课题**: 如何将洞察转化为行动

**机会路径**: 你的眼光是稀缺的。先从一个小的验证开始，让洞察落地。

**具体动作**:
1. 选定一个最看好的目标市场深入研究
2. 找一家已经在当地做好的供应商合作验证
3. 用最小成本测试市场反应

**行动包**: 《目标市场深度调研模板》、《海外供应商合作指南》、《小成本市场验证方案》

---

### C. 破局先锋 🚀

**称号**: 一往无前

**summary**: 行动至上，说到做到

**特征描述**:
> 你是一个行动派。想清楚了就干，不犹豫、不拖延、不观望。你相信出海这件事，想太多没用，先干了再说。你愿意尝试新的平台、接触新的客户、接受新的挑战。你的魄力和勇气是你最大的武器，很多机会就是被你这样敢冲的人抓住的。

**优势**:
- 执行力超强
- 敢于尝试新事物
- 不怕失败
- 行动速度快

**天赋课题**: 如何让行动更有成效

**机会路径**: 你的行动力是最大的优势。学会在行动中调整，让每一步都更有价值。

**具体动作**:
1. 设定每个行动的明确目标
2. 建立行动复盘机制
3. 找到适合自己的出海节奏

**行动包**: 《出海行动规划模板》、《执行复盘表格》、《出海节奏指南》

---

### D. 谋定后动 🧮

**称号**: 深谋远虑

**summary**: 三思后行，稳中求胜

**特征描述**:
> 你是一个相信逻辑和数据的人。在做任何决定之前，你都会把所有的情况想清楚，把成本收益算明白。你相信成功的出海不是靠运气，而是靠周密的计划。你不喜欢打无准备的仗，每一步都要心中有数。这种特质让你很少犯错，但有时候也会错过一些稍纵即逝的机会。

**优势**:
- 决策有理有据
- 风险意识强
- 计划性强
- 善于分析

**天赋课题**: 如何在谨慎和敏捷之间找到平衡

**机会路径**: 你的深思熟虑是宝贵的能力。给自己设定一个决策时限，让计划落地。

**具体动作**:
1. 为出海设定明确的决策时限
2. 先用小成本验证，大决策分段做
3. 接受"足够好"而非"完美"

**行动包**: 《出海决策框架》、《小步验证方法论》、《决策时限设定指南》

### E. 六边形战士 ⭐

**称号**: 六边形战士

**summary**: 全面均衡，天时地利

**特征描述**:
> 在所有测评者中，你是难得的全面型选手。产能稳定、市场洞察清晰、执行力到位、决策也理性有据。你可能是见证过工厂从小做到大的传承者，也可能是不断学习提升的进取者。你的综合实力已经超越了大多数同行，现在需要的只是一个契机，和踏出第一步的决心。

**优势**:
- 能力全面均衡
- 综合实力强
- 适应能力好
- 抗风险能力强

**天赋课题**: 如何突破自我设限

**机会路径**: 你已经站在了起跑线的前面。差的就是那临门一脚。

**具体动作**:
1. 选定一个目标市场和渠道
2. 找一家靠谱的服务商加速
3. 设定3个月的第一个里程碑

**行动包**: 《出海加速计划》、《服务商选择白名单》、《里程碑规划模板》

### F. 蓄势待发 🤔

**称号**: 厚积薄发

**summary**: 积累沉淀，静待破茧

**特征描述**:
> 你现在的状态是「万事俱备，只欠东风」。你有一点基础，有一点认知，做过一些准备，但总觉得还差一点什么。你不急着冲动行事，但也没有停下脚步。你相信好的时机比盲目的行动更重要，在等待中积累力量。每一个成功出海的工厂，都经历过这样的蓄力期。

**优势**:
- 心态稳健
- 愿意学习
- 善于积累
- 脚踏实地

**天赋课题**: 如何找到突破的契机

**机会路径**: 你需要的不是更多准备，是一个开始的理由。现在就是最好的时机。

**具体动作**:
1. 梳理你已经拥有的资源
2. 找一个可以马上开始的最小行动
3. 找一个同行者一起出发

**行动包**: 《出海准备度评估表》、《最小行动清单》、《出海伙伴对接服务》

### G. 敏捷猎手 💨

**称号**: 逐风捕影

**summary**: 嗅觉敏锐，快速响应

**特征描述**:
> 你有一种猎手般的敏锐。看到市场机会，你会第一时间响应；发现新的平台，你会马上去研究；嗅到客户需求，你会立刻去跟进。你的反应速度很快，学习能力也很强，能够快速抓住转瞬即逝的机会。这种敏捷是你最大的优势，但也需要学会在快速行动中保持专注。

**优势**:
- 市场嗅觉灵敏
- 学习能力强
- 适应变化快
- 执行效率高

**天赋课题**: 如何在敏捷中保持专注

**机会路径**: 你的敏捷是稀缺能力。找到最值得投入的方向，深耕下去。

**具体动作**:
1. 锁定1-2个核心机会
2. 深入研究而非广泛撒网
3. 建立自己的核心优势

**行动包**: 《机会筛选框架》、《深度研究方法论》、《核心优势定位表》

**优势**:
- 战略眼光强
- 规划能力出众
- 知识储备丰富

**天赋课题**: 如何让规划落地为行动

2. 先用一个产品、一个市场试水

**行动包**: 《低风险出海方案》、《出海服务商选择指南》、《代运营合作模板》

### M. 精算智多星 📊

**称号**: 明察秋毫

**summary**: 精打细算，理性制胜

**特征描述**:
> 你是一个相信数据和逻辑的人。在做任何决定之前，你都会把账算得清清楚楚——投入多少、预期回报、风险几何，你都要心中有数。你不相信运气，只相信数据；不追求暴富，只追求稳健。这种理性的特质让你很少做出冲动的决定，但有时候也会因为过度分析而错失良机。

**优势**:
- 数据分析能力强
- 成本意识强
- 决策理性
- 风险可控

**天赋课题**: 如何在理性中保持敏锐

**机会路径**: 你的理性是宝贵的资产。学会在数据不完整时也能做出决策。

**具体动作**:
1. 设定可接受的风险底线
2. 用小步快跑代替大而全的分析
3. 接受"足够好的决策"

**行动包**: 《理性决策框架》、《小步验证方法》、《容错心态培养》
